# Chat Application

## Description

Cette application de chat en temps réel utilise WebSocket et le protocole STOMP pour permettre la communication entre les clients. Les utilisateurs peuvent entrer leur nom, taper des messages et les envoyer à tous les autres utilisateurs connectés. Les messages sont affichés en temps réel dans la liste des messages.

## Structure du Code

### `index.html`

Le fichier `index.html` est la base de l'application web. Il inclut le HTML pour l'interface utilisateur, ainsi que le JavaScript nécessaire pour gérer les interactions en temps réel via WebSocket.
